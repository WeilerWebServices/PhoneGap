// Intro + About

We’re celebrating all things PhoneGap and want you to join us! Come to PhoneGap Day in Amsterdam on Tuesday September 24, 2013.

Part conference, part celebration, expect a fun and informative day packed with people who are using PhoneGap as part of their mobile app development. Meet members of the PhoneGap team and connect with other PhoneGap developers. You’ll hear about the latest PhoneGap news and what’s coming next!

// Workshops

Dive deeper into PhoneGap by attending our pre-conference workshops on Monday, September 24. Choose between two different tracks:

PhoneGap, from beginner to God Mode [SOLD OUT]


  • Beginner: Building your first PhoneGap app
    Workshops led by Christophe Coenraets and Holly Schinsky
    This workshop will cover everything that you need to know to get started using PhoneGap. It will start at the conceptual level – what is PhoneGap, will then move on to showcase sample phonegap applications, discuss development and debugging options, and highlight common development frameworks. Finally, the presentation will end with a walkthrough of creating “your first” PhoneGap application using the command line tools, and with PhoneGap Build.
  • Advanced: Architecting PhoneGap apps
    Workshops led by Christophe Coenraets
    Learn how to architect large, complex, and native-like PhoneGap apps using HTML, JavaScript, and CSS. We investigate the Single Page Architecture, HTML templates, effective Touch events, performance techniques, MVC Frameworks and more.
  • God Mode: Extending PhoneGap with Plugins and Supporting Push Notifications
    Workshops led by Holly Schinsky
    Learn how to take your application to the next level by extending PhoneGap with plugins and other advanced features such as push notifications. In this session we'll walk through the plugin API and develop, configure and use our own simple plugin in a PhoneGap application. We'll also learn how to set up our application to receive push notifications using the PushPlugin supported by PhoneGap Build.

PhoneGap Ecosystem: the view from Google and Microsoft [SOLD OUT]


  • Learn everything about Chrome Packaged Apps with Google and Cordova. Build HTML5 apps that run offline by default, access the native platform, and provide the user with an immersive experience. Chrome Packaged Apps run on 4 desktop and 2 mobile OSes right out of the box, with the skills you already have today.
  • Learn about Windows Phone 8 tools and support for Cordova with Microsoft and community experts. Have a PhoneGap app? Find out how to migrate your PhoneGap app to the Windows Phone 8 store in a few hours.
